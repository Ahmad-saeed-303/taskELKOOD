
# Task: ELkood Job 

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone <repo>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run start
   ```

4. Create accounts:
   - One for a user.
   - One for an entity (company).

5. Log in as an entity to:
   - Post new jobs.
   - Manage existing ones.

6. Log in as a user to:
   - Browse available jobs.
   - Apply for a job.
```
